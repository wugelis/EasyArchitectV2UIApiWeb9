## 前言
近年來，從前年 2022年11月30號 ChatGPT 推出以來，可說席捲整個世界，生成式 AI 將取代你的工作一詞不斷的在新聞媒體版面、各行各業、甚至你生活周遭出現，不亞於當年 iPhone 的推出，但我覺得更甚於此，因為 AIGC 的出現，讓我原先沒有特別看好的 LLM 大型語言模型原來能夠 Pre-Trained 到這樣的地步，類神經網路居然能將自然語言（NLP）的處裡，表達做到基本邏輯推演甚至通過圖靈測試的地步，這讓許多人、各行各業、開發人員都燃起盞明燈。而取代之說，也如雨後春筍般地湧現，是好是壞大家不僅開始思考各種因應的對策與方法。但趨勢通常都不可檔，只能去理解它、瞭解它、利用它、接受它、擁抱它、未來它會成為我們生活周遭的一部份。
